## 快捷键
```
cmd＋o打开
Z 选框放大
按住空格 平移
cmd＋0 全图显示
cmd＋1 原图显示
cmd＋＋ 放大
cmd＋－缩小
cmd＋d 取消选择
cmd＋t 自由变换
cmd＋z 还原
[    ]  画笔大小调整
alt＋del 前景色填充
cmd＋del 背景色填充
```

## 图层
```
cmd＋j 复制图层
cmd＋e 合并图层
shift＋cmd＋e 合并可见图层
cmd＋shift＋alt＋e 盖印图层
移动 链接图层
新建图层组 删除图层组
```
