# 开发工具常用快捷键
<http://www.wxapp-union.com/forum.php?mod=viewthread&tid=2141>
1: opt + shift + f : 代码格式化
2: cmd + [, cmd+ ]代码行缩进
3: cmd + shift + [, cmd + shift +] 折叠打开代码块
4: cmd + /shift +  return 在当前行的 上方/下方 插入一行
5: shift + opt + up/dow, 向上向下复制一行
6: opt + up/dow 上下移动一行
7: cmd + i 选中当前行
8: cmd + s保存
9: cmd + return  移动到文件结尾
10: cmd + r 运行程序
