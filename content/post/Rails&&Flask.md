# 主要的异同
rails 重量级 基于ruby
flask 轻量级 基于python
都是web开发框架。

# rails
熟悉（工作需要）

rails是一个重型的开发武器。给你提供了各种开发时候的需求。
你想要的模块基本上都内置了。



# flask
了解（个人兴趣）
flask是一个轻量的开发框架。有几个类似市场一样的东西。其实也不是。
flask-* 为你提供了你可能想要开发的模块。

# 总结
**殊途同归**
统一问题的不同解决方式。
