# Hexo
hexo是什么？
快速、简洁且高效的博客框架
但是就看你怎么用了。
今天我打算把它用为我的公众号和小程序的**数据源**
如果只是搭建博客。就不用深入的理解他了。

但是如果想用的好，做点其他用途就要好好思量一下了。

# 快速搭建博客
```shell
npm install hexo-cli -g
hexo init blog
cd blog
npm install
hexo server
```

# Hexo的特点
1. 生成速度快
2. 支持MD
3. 一件部署
4. 丰富的插件
