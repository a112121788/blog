# ImageView
ImageView: Displays an arbitrary image
显示一个任意的 image
ImageView.ScaleType : Options for scaling the bounds of an image to the bounds of this view.

# 个人总结
显示一个图片(能访问到的，本地、网络、内存等地方)
核心点：ScaleType
也就是说: 图片的原始尺寸和给定的框框 进行怎样的处理能达到一个最优（最符合我们需求）的显示。
